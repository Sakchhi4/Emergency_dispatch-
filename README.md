# Emergency Dispatch System

A Java-based emergency response system that manages incidents, ambulances, responders, and dispatch operations using JDBC and MySQL.

# Features

* Incident logging & tracking
* Ambulance & responder management
* Automated dispatch system
* Patient, equipment & triage records
* Status updates & reports

# Concepts Used

* CRUD operations
* JDBC (MySQL)
* OOP: Encapsulation, Inheritance, Polymorphism, Abstraction
* Interface (`CrudOperations`) & Abstract Class (`BaseDAO`)
* Input validation

# Tech Stack

* Java
* MySQL
* JDBC

# Run

```bash
javac -cp ".;lib/mysql-connector-j-9.x.x.jar" EmergencyDispatchApp.java
java -cp ".;lib/mysql-connector-j-9.x.x.jar" EmergencyDispatchApp
```
